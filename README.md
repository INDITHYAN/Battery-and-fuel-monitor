# ⚡ Multi-Mode Fuel and Battery Monitoring System

## 🚀 Overview
A smart monitoring system that tracks fuel and battery levels in real-time across multiple modes such as electric vehicles, hybrid systems, and generators.

## 🔥 Features
- Ultrasonic-based fuel level detection
- Battery level monitoring
- Multi-mode operation (EV, Hybrid, Generator)
- LCD display output
- PC dashboard visualization
- Data logging (CSV / SQLite)
- Alert system for low fuel

## 🧠 Working
- Ultrasonic sensor measures fuel level
- Potentiometer simulates battery level
- Arduino processes data using C
- Data is displayed on LCD and PC dashboard
- Alerts trigger when fuel < 10%

## 🛠️ Components Used
- Arduino Nano
- HC-SR04 Ultrasonic Sensor
- Potentiometer
- I2C LCD
- Serial Communication
